# Fear vs Greed Trading Analysis

## Project Overview
This project analyzes the impact of market sentiment (Fear vs Greed) on trading performance.  
It combines sentiment data from the Fear & Greed Index with historical trading data to calculate metrics like:

- Average PnL (Profit & Loss) per sentiment  
- Win rate comparison  
- Impact of trade size and trader consistency  
- Performance of frequent vs rare traders  

The analysis is performed using Python, pandas, numpy, and matplotlib.
